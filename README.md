# teacher-ai


## 介绍

用于教师的AI工具，尤其适用于语文老师，可以一键批量给试卷打分，再也不用为学生龙飞凤舞的字体苦恼。技术栈：文心一言、百度识图、tauri、react

##　开发

```bash
# 安装依赖
pnpm install

# 启动开发
pnpm dev
```

## 打包

```bash
# 打包
pnpm build
```
